# Smart Task Planner

UI/UX prototype for a task management platform.

## Website Name
SmartTask

## Project Description

We developed "SmartTask" to help individuals manage their daily goals and deadlines in one place. The platform solves the problem of task scattering by providing a clear dashboard with priority tools.

## Features

- Create new tasks
- Edit existing tasks
- Delete tasks
- Assign priority levels (High, Medium, Low)
- View tasks in an organized list
- Mark tasks as completed
- Dynamic task management using JavaScript
- Due date selection
- Daily tasks with excluded days
- Goal linking
- Local storage support
- Support form validation
- Responsive dark UI design
